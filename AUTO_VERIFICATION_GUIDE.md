# 자동 인증 시스템 가이드

## 개요

99% 정확도의 다단계 자동 인증 시스템으로, 인플루언서와 유명인을 즉시 검증합니다.

## 점수 시스템 (총 100점)

### 1. 소셜 미디어 검증 (30점)
- **Instagram (15점)**
  - 인증 배지(블루체크): 5점
  - 1만 팔로워 이상: 5점
  - 10만 팔로워 이상: 추가 3점
  - 계정 생성 6개월 이상: 2점

- **YouTube (10점)**
  - 인증 배지: 3점
  - 1만 구독자 이상: 4점
  - 10만 구독자 이상: 추가 3점

- **Twitter (5점)**
  - 인증 배지: 3점
  - 1만 팔로워 이상: 2점

### 2. 웹 존재성 검증 (25점)
- **Google 검색 결과 (15점)**
  - 1000건 이상: 10점
  - 500-999건: 7점
  - 100-499건: 5점
  - 10-99건: 3점

- **뉴스 기사 언급 (10점)**
  - 50건 이상: 10점
  - 20-49건: 7점
  - 10-19건: 5점
  - 5-9건: 3점

### 3. 활동성 검증 (20점)
- 최근 게시물/영상 활동
- 참여도(좋아요, 댓글 비율)
- 팔로워 수 기반 추가 점수

### 4. 일관성 검증 (15점)
- 3개 이상 플랫폼에서 활동: 10점
- 2개 플랫폼: 7점
- 1개 플랫폼: 4점
- 프로필 정보 완성도: 5점

### 5. 신뢰도 검증 (10점)
- 상세 설명 제공: 5점
- 뉴스 검색 키워드 제공: 3점
- 카테고리 및 직업 정보: 2점

## 판정 기준

- **90점 이상**: 🎉 **자동 승인** (즉시 인증)
- **70-89점**: ⚠️ **관리자 검토 필요** (1-3일)
- **70점 미만**: ❌ **자동 거부** (재신청 가능)

## API 키 설정 (99% 정확도 달성)

정확도를 최대화하려면 다음 API 키를 설정하세요:

### 1. YouTube Data API v3
```
발급: https://console.cloud.google.com/apis/credentials
.env: YOUTUBE_API_KEY=your_key_here
```

### 2. Twitter API v2
```
발급: https://developer.twitter.com/
.env: TWITTER_BEARER_TOKEN=your_token_here
```

### 3. Google Custom Search API
```
발급: https://console.cloud.google.com/apis/credentials
.env: GOOGLE_API_KEY=your_key_here
     GOOGLE_CSE_ID=your_cse_id_here
```

### 4. News API (이미 설정됨)
```
발급: https://newsapi.org/
.env: NEWS_API_KEY=a6ad62efa8f44721958d7eedbb313217
```

## 사용 방법

### 사용자
1. 메뉴 → "본인 인증 요청"
2. 인증 유형 선택 (연예인, 인플루언서 등)
3. 정보 입력:
   - **필수**: 실명, 직업
   - **권장**: Instagram, YouTube, Twitter 링크
   - **선택**: 팔로워 수, 뉴스 키워드
4. 제출 → 즉시 자동 검증 실행

### 결과
- **자동 승인**: 즉시 ✓ 배지 표시
- **검토 필요**: 관리자 검토 대기
- **자동 거부**: 점수 확인 후 재신청

### 관리자
1. 메뉴 → "인증 관리"
2. 자동 검증 점수 확인
3. 필요시 수동 승인/거부

## 점수 향상 팁

### 90점+ 자동 승인 받는 법

1. **소셜 미디어 최적화**
   - Instagram, YouTube, Twitter 모두 등록
   - 팔로워 수 1만 이상 유지
   - 인증 배지(블루체크) 획득

2. **웹 존재성 강화**
   - 뉴스 기사, 인터뷰 활동
   - 공식 웹사이트 운영
   - 위키피디아 페이지 생성

3. **일관성 유지**
   - 모든 플랫폼에서 동일한 이름 사용
   - 프로필 사진, 설명 일치

4. **상세 정보 제공**
   - 뉴스 검색 키워드 입력
   - 상세한 설명 작성
   - 카테고리 정확히 선택

## 예시

### 자동 승인 사례 (95점)
```javascript
{
  realName: "아이유",
  occupation: "가수",
  category: "연예인",
  followerCount: 500000,
  socialLinks: {
    instagram: "https://instagram.com/dlwlrma",  // 10만+ 팔로워
    youtube: "https://youtube.com/@IU",           // 10만+ 구독자
    twitter: "https://twitter.com/IU_official"    // 1만+ 팔로워
  },
  newsKeywords: "아이유 가수, IU, 이지은",
  description: "가수 겸 배우로 활동 중..."
}

점수 상세:
• 소셜 미디어: 28/30 (Instagram 15, YouTube 10, Twitter 3)
• 웹 존재성: 25/25 (Google 5000+ 결과, 뉴스 100+ 기사)
• 활동성: 20/20 (최근 게시물 활발)
• 일관성: 15/15 (3개 플랫폼, 일관된 정보)
• 신뢰도: 10/10 (완전한 정보)

총점: 98/100 → ✓ 자동 승인
```

### 검토 필요 사례 (75점)
```javascript
{
  realName: "홍길동",
  occupation: "인플루언서",

점수 상세:
• 소셜 미디어: 10/30 (Instagram만, 인증 배지 없음)
• 웹 존재성: 12/25 (Google 100+ 결과, 뉴스 5건)
• 활동성: 15/20 (팔로워 1만+)
• 일관성: 9/15 (1개 플랫폼만)
• 신뢰도: 6/10 (기본 정보만)

총점: 52/100 → ⚠️ 관리자 검토 필요
```

## 문제 해결

### API 키 없이도 작동하나요?
네, 하지만 정확도가 낮아집니다 (약 60-70%).
- 팔로워 수 기반 기본 점수 부여
- 제출된 정보 일관성 검증
- 대부분 "관리자 검토 필요"로 판정

### 점수를 높이려면?
1. 더 많은 소셜 플랫폼 등록
2. 팔로워 수 증가
3. 뉴스 기사 활동 증가
4. 상세한 정보 제공

### 재신청 가능한가요?
네, 거부 후 언제든 재신청 가능합니다.
점수를 높인 후 다시 시도하세요.
